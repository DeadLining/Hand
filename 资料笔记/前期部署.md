### IDEA服务的前期配置

1. 更改 application.yml

   ```yaml
   spring:
     sleuth:
       sampler:
         percentage: 1.0
     application:
       name: hzero-cms-28250
   ```

2. copy **XXXExtraDataManager** to **LocalXXXExtraDataManager**

   ```java
   /*XXXExtraDataManager*/
   routeData.setName("hcms");
   routeData.setPath("/hcms/**");
   routeData.setServiceId("${hzero.service.platform.name:hzero-cms}");
   /*LocalXXXExtraDataManager*/
   routeData.setName("hcms-28250");
   routeData.setPath("/hcms-28250/**");
   routeData.setServiceId("${hzero.service.platform.name:hzero-cms-28250}");
   ```

3. Edit Configurations -> Active profiles: local
