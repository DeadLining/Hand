### Ribbon负载均衡

```java
@Bean
@LoadBalanced
RestTemplate template(){
    return new RestTemplate;
}
```

### Feign声明式的伪Http客户端

Feign默认集成了Ribbon，并和Eureka结合，默认实现了负载均衡的效果

面向接口：采用的是基于接口的注解

- 完全不管实现
- 传入规定格式的数据即可

---

### Hystrix熔断器

```yaml
//开启
feign:
  hystrix:
    enabled: true
```

```java
@FeignClient(value = "hzero-iam", fallback = IamRemoteHystrix.class)
```

### zuul

SpringCloud网关接口就类似于转发

```yaml
//配置文件：转发地址
zuul:
  routes:
    api-a:
      path: /api-a/**
      serviceId: XXX-ribbon
    api-b:
      path: /api-b/**
      serviceId: XXX-feign
```

```java
@EnableEurekaClient
@EnableZuulProxy
```
