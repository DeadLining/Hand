### git远程仓库与本地仓库之间的交流

条件：windows凭据（路径：`控制面板->用户和账户->管理凭据`）

windows凭据可以保存之前的会话账号、密码等登录信息

代码实现：

> git remote add origin http://XXXXXXXX.git
>
> git add
>
> git commit -m "msg"
>
> git push -u origin master

代码解析：

> git remote add origin(别名) http://XXXXXXXX.git(远程仓库地址)
>
> git add
>
> git commit -m "msg"
>
> git push -u(远程与本地相连接) origin master(提交更新至origin(远程仓库)的master分支中)

**注：**若push的远程仓库的分支目标不存在，则远程仓库会<font color='red'>自动创建</font>指定的分支

##### Create a new repository

```
git clone https://code.choerodon.com.cn/28250/kong.git
cd kong
touch README.md
git add README.md
git commit -m "add README"
git push -u origin master
```

##### Existing folder

```
cd existing_folder
git init
git remote add origin https://code.choerodon.com.cn/28250/kong.git
git add .
git commit -m "Initial commit"
git push -u origin master
```

##### Existing Git repository

```
cd existing_repo
git remote rename origin old-origin
git remote add origin https://code.choerodon.com.cn/28250/kong.git
git push -u origin --all
git push -u origin --tags
```
